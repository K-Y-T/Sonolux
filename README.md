# Sonolux

Sonolux is an ongoing personal engineering project exploring the intersection of analogue electronics, embedded systems, signal processing, and software for music visualisation.

Originally inspired by a university assignment involving a three-band analogue frequency display, the project has since evolved into a broader platform for developing hardware and software related to audio signal analysis and visualisation.

---

## Current Status

The repository currently contains:

- Sound2Light prototype (breadboard implementation)
- Sonolux V1 analogue hardware design
- Circuit schematics
- PCB design files

The project is under active development and will continue to evolve throughout my undergraduate studies.

---

## Project Goals

Current long-term objectives include:

- Five-band analogue frequency visualisation
- Analogue Speaker system
- Custom PCB design
- Embedded control and power management
- Bluetooth audio integration
- JUCE-based desktop software
